CODE

The name of the game is Drone-Aid.The objective of the game is to use a drone to deliver materials remote island in the Asian Pacific that was just affected by an earthquake. Delivering the materials will be tricky, there are tall cell towers near the drop off loctions with the highest populations. You will need to avaid the cell towers or the drones will crash into them or be damaged. The mission is to deliver as many materials to the island while avoiding the cell towers. Your goal is to aid at least 1300 people. If you hit a cell tower it will negatively impact your score and the amount of people you have aided because they will have to use the materials to fix the cell towers instead of supplying and repairing the island. This game was developed using HTML, CSS and Javascript.

Weather Wise- This App is the Craigslist of Weather. It is designed to be an outlet for crowdsourcing information that relates weather patterns and events real-time by an exact location. Users can share actual weather events by state, city, county, neighborhood and even street, so other users can become wise about the weather, and can make wiser decisions to keep them and their families safe. This project was created using MongoDB, React.js, Node.js and Express.

Nightly- This project was created to connect people with shelter resources nightly. Unbooked Hotel rooms are donated to individuals in need of shelter in exchange for a services provided at the hotel. Examples of services provided are cleaning the parking lot or exterior windows in exchange for a nights stay. Guest choose the service they will provide while booking. There is a page where all guests, hotels and services on the platform can be viewed. This project was created using MongoDB, Handlebars, Node.js and Express.
